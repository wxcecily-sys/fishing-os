# Fishing OS — 规格说明书 v1.0

## 概述

**项目名称**: Fishing OS  
**版本**: 1.0  
**基于**: Ubuntu 22.04 LTS  
**类型**: 桌面操作系统  
**目标大小**: ~2GB (单 ISO)  
**目标用户**: 普通日常用户、开发者、游戏玩家  

---

## 系统架构

```
┌──────────────────────────────────────────────────────┐
│                   Fishing OS                        │
├──────────────────────────────────────────────────────┤
│  桌面环境: Xfce4 (轻量化)          ~400MB           │
│  Wine 6.0 + Bottles 兼容层          ~300MB           │
│  Ollama 本地 AI 引擎                ~200MB (不含模型) │
│  MCP Server (内置助手)              ~50MB             │
│  Firefox ESR 浏览器                  ~150MB           │
│  LibreOffice 办公套件                ~300MB           │
│  VLC 多媒体播放器                    ~100MB           │
│  游戏工具 (Steam, GameMode)         ~150MB           │
│  开发工具 (VS Code, Git, Python)     ~200MB           │
│  系统基础 + 主题美化                 ~200MB           │
│  预留空间 (用户可用)                ~950MB           │
├──────────────────────────────────────────────────────┤
│              总计 ≈ 2GB                             │
└──────────────────────────────────────────────────────┘
```

## 核心组件

### 1. Linux 基础
- **内核**: Ubuntu 22.04 LTS (Jammy)
- **架构**: x86_64
- **桌面环境**: Xfce4 (轻量级，符合 2GB 限制)
- **显示管理器**: LightDM
- **文件系统**: ext4

### 2. Wine Windows 兼容层
- **Wine 版本**: Wine HQ 8.0 (Staging)
- **前端工具**: Bottles (图形化 wine 管理)
- **预配置**: 常见 Windows 运行环境
- **GameMode**: 自动游戏优化

### 3. MCP 本地 AI 助手
- **AI 引擎**: Ollama (完全离线)
- **默认模型**: Qwen2.5 3B (轻量，适合 2GB 系统) 或 llama3.2 1B
- **MCP 协议**: 模型上下文协议
- **能力**:
  - 文件操作 (创建/读取/移动/搜索)
  - 命令执行
  - 联网搜索 (可选，本地模型不支持时跳过)
  - 代码生成与解释
  - 自然语言对话
  - 任务提醒

### 4. 预装软件

| 类别 | 软件 | 用途 |
|------|------|------|
| 浏览器 | Firefox ESR | 网页浏览 |
| 办公 | LibreOffice 7.x | 文档/表格/演示 |
| 媒体 | VLC | 视频/音频播放 |
| 开发 | VS Code | 代码编辑 |
| 开发 | Git + GitHub CLI | 版本控制 |
| 开发 | Python 3.10 + pip | 开发环境 |
| 游戏 | Steam + Proton | 游戏平台 |
| 游戏 | GameMode | 游戏优化 |
| 工具 | GParted | 分区管理 |
| 工具 | Synaptic | 软件包管理 |

### 5. 游戏支持
- **Steam** (通过 Proton 运行 Windows 游戏)
- **GameMode** (自动优化 CPU/GPU)
- **预配置 Proton 兼容层**
- **常见游戏依赖库**

## 构建流程

### 阶段 1: 基础系统 (在宿主 Ubuntu 22.04 上操作)

1. 安装基础工具
2. 安装 Ubuntu Base System
3. 配置软件源
4. 安装桌面环境

### 阶段 2: 软件包安装

1. 安装 Xfce4 + LightDM
2. 安装 Wine + Bottles
3. 安装 Ollama
4. 安装 Firefox ESR
5. 安装 LibreOffice
6. 安装 VLC
7. 安装开发工具
8. 安装 Steam + GameMode

### 阶段 3: MCP 助手集成

1. 配置 Ollama 服务
2. 安装 MCP Server
3. 配置 AI 助手界面
4. 创建桌面启动器

### 阶段 4: 美化与优化

1. 应用自定义主题
2. 配置壁纸
3. 创建桌面图标
4. 清理缓存
5. 优化 ISO 大小

### 阶段 5: 打包

1. 使用 Cubic 定制 ISO
2. 配置引导加载器
3. 生成最终 ISO
4. 测试启动

## 技术规范

### 硬件要求
- **CPU**: 64位 x86 处理器 (双核推荐)
- **内存**: 4GB RAM (推荐 8GB)
- **硬盘**: 至少 4GB 可用空间
- **显卡**: 支持 OpenGL 3.3+

### ISO 规范
- **文件名**: `fishing-os-1.0-amd64.iso`
- **大小限制**: < 2.1GB
- **引导**: BIOS + UEFI 双支持
- **语言**: 中文 + 英文
